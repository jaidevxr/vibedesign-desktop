# MindSync — AI Wellness Companion

A mobile-first mindfulness app with real AI-powered therapy chat, guided breathing exercises, mood tracking, and personalized wellness insights.

## Features
- 🧠 **AI Therapist** — Real-time chat powered by Groq (Llama 3.3 70B)
- 🧘 **Guided Meditation** — Box Breathing, 4-7-8, and Calm Breathing exercises
- 📊 **Stats Dashboard** — Track mood, energy, sessions with Recharts visualizations
- 🔥 **Streak Tracking** — Automatic daily streak calculation
- 👤 **Profile & Settings** — Persistent preferences via localStorage

## Tech Stack
- React 18 + TypeScript + Vite
- Tailwind CSS + Custom glassmorphism design system
- Recharts for data visualization
- Groq API for AI chat (Llama 3.3 70B)

## Getting Started
```bash
npm install
npm run dev
```

Open http://localhost:8080 in your browser.
